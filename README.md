## Features

>### P0
- Create, Edit, and delete posts.


- Filter notes by trending (most likes), and sort by lastest .

- Comment on posts

- Bookmark posts

- Follow / Unfollow user

- Edit user profile , add / change profile picture.

>### P1

- Share pictures in posts. 

- Add background image.
